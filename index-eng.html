Vacancy Management:

  • Create a vacancy with pre-made templates
  • Make a preliminary schedule for posting and closing the vacancy
  • Ability to post internal and external vacancies
  • Define tags for a vacancy and search with tags
  • Create an individual application form for vacancies
  • Post vacancies with one click jobs.ge, hr.ge, candidate.ge
  • Share a vacancy on a social network
  • Vacancy cloning
  • Notifications of activities about the vacancy, such as: submitting application, moving to stages, etc.

Application management:

  • Defining individual stages
  • Monitoring of received applications
  • Search for specific keywords in received applications

Candidate Management:

  • Define tags for the candidate and search with tags
  • Sharing the candidate for an outsider
  • Profile editing by the recruiter
  • Join a candidate in a pre-created group
  • Disqualification of the candidate
  • Fix the reasons for disqualification
  • Add a specific candidate to the vacancy
  • View all vacancies for which the candidate has applied
  • Make a note, send an email, schedule meetings, make assessments, attach work files
